# GeeksforGeeks Solutions 🚀

This repository automatically syncs my accepted solutions from GeeksforGeeks using a browser extension.

## Setup Instructions

To sync your GFG activity to this repository, follow these steps:

1. **Install the Extension:**
   - Install the **"GfG to GitHub"** extension from the [Chrome Web Store](https://chrome.google.com/webstore/detail/gfg-to-github/gojabhkegjnlnklkkpkglaembhlknkgk).

2. **Authenticate with GitHub:**
   - Open the extension and click **Authenticate**.
   - Grant it access to manage your repositories.

3. **Link this Repository:**
   - Once authenticated, select this repository (**`gfg-solutions`**) from the dropdown menu in the extension.
   - Click **Get Started**.

4. **Start Solving!**
   - Every time you solve a problem on GeeksforGeeks and click **Submit**, the extension will automatically push your code to this repository, organized by difficulty level.
