# GeeksforGeeks Solutions 🚀

This repository automatically syncs my accepted solutions from GeeksforGeeks using a custom GitHub Actions workflow.

## How it Works
- The sync runs daily at midnight UTC via GitHub Actions.
- It uses a Python script (`gfg_sync.py`) to fetch solved problems.
- Solutions are organized by difficulty level.

## Setup Instructions

To enable automatic syncing, you need to add your GeeksforGeeks details as GitHub Secrets:

1. **Go to Settings:**
   - In this repository, go to **Settings** > **Secrets and variables** > **Actions**.

2. **Add New Secrets:**
   - Click **New repository secret** and add:
     - Name: `GFG_USERNAME` | Value: (Your GFG handle, e.g., `laksh1411`)
     - Name: `GFG_COOKIE` | Value: (Your full session cookie string)

3. **How to find your Cookie:**
   - Log in to [GeeksforGeeks](https://www.geeksforgeeks.org/).
   - Press **F12** > **Application** tab > **Cookies** > Select `https://www.geeksforgeeks.org`.
   - Copy the entire "Value" for any key like `session_id` or just copy the whole `cookie` header from a network request.

4. **Enable Permissions:**
   - Go to **Settings** > **Actions** > **General**.
   - Change **Workflow permissions** to **"Read and write permissions"** and click **Save**.

5. **Run Manually (Optional):**
   - Go to the **Actions** tab, select **GFG Sync**, and click **Run workflow** to sync immediately.
